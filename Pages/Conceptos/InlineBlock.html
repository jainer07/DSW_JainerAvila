Valor block, inline e inline-block

Todos los elementos tienen un valor display predeterminado según el tipo de elemento, el valor predeterminado para la mayoría de los elementos suele ser block o inline

Uso del valor block

Ejemplo

Resultado

El elemento genera un cuadro de elemento de bloque y ocupa todo el espasio posible.

La forma de usarlo es: propiedad valor

display: block

<div style="background: black; padding: 5px;">

<h2 style=" display: block; border: 4px dotted #fff; color:#fff">Elemento de bloque 1 </h2> <h2 style=" display: block; border: 4px dotted #fff; color:#fff">Elemento de bloque 2 </h2>

</div>

Elemento de bloque 2

Uso del valor inline

Ejemplo

Resultado

El elemento genera uno o más cuadros de elemento en línea y ocupa el espacio que el elemento necesita.

La forma de usarlo es: propiedad valor

display: inline

<div style="background: black; padding: 5px;">

<h2 style=" display: inline; border: 4px dotted #fff; color:#fff">elemento 1 </h2> <h2 style=" display: inline; border: 4px dotted #fff; color:#fff">elemento 2 </h2> <h2 style=" display: inline; border: 4px dotted #fff; color:#fff">elemento 3 </h2>

</div>

Uso del valor inline-block

Ejemplo

Resultado

La forma de usarlo es: propiedad valor

display: inline-block
