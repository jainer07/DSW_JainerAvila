Propiedad Display en los elementos

Todos los elementos tienen un valor display predeterminado según el tipo de elemento, el valor predeterminado para la mayoría de los elementos suele ser block o inline


Conocer los valores de la propiedad display

  • block
  • contents
  • flex
  • flow-root
  • grid
  • inherit
  • inline-block
  • inline.flex
  • inline-grid
  • inline-table
  • list-item
  • none
  • rever
  • table
  • table-caption
  • table-cell
  • table-column
  • table-column-group
  • table-footer-group
  • table-header-group
  • table-row
  • table-row-group
  • unset
  • --webkit-box
  • --webkit-inline-box

Usos del valor block

El elemento genera un cuadro de elemento de bloque y ocoupa todo espaso posible.


Usos del valor inline

El elemento genera uno o más cuadros de elemento en línea y ocupa el espacio que el elemento necesita.


Usos del valor inline-block

es una combinación entre los dos valores anteriores, crea cajas que son de bloque y en línea de forma simultánea,Una caja de tipo inline-block se comporta como si fuera de bloque, pero respecto a los elementos que la rodean es una caja en línea.
